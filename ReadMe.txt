msRun.m will walk you through the order one would generally run the processing and analysis functions on a single recording session. We highly recommend starting here when working with this analysis package for the first time.

msBatchRun.m outlines the order one would generally run the processing and analysis functions on a batch of recording sessions.

The SignleFunctions folder contains all analysis and processing function that run on a single data set stored in the MATLAB workspace.

The BatchFunctions folder contains functions that will run analysis and processing on all data sets located in the folder and subfolders of the data path sent in the functions argument.
